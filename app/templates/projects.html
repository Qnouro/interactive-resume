Bot Dylan

Trained recurrent neural networks on google colaboratory in order to generate song lyrics based on an artist's style. The data was scraped using the Genius Api and publicly available books.
Tools: Python - PyTorch - Pandas - Genius API - Google Colab (GPU).

World Events Visualization

Created a data pipeline from the Gdelt project that automatically updates a responsive world map with the associated news sources. The map is restarted every day.
Tools: Python - Pandas - SQL - D3.js.

Leaf Classifier

Trained a leaf classification model to determine the family of a leaf based on a grayscale image. The data was manually scraped from PlantMethods and cleaned using grabcut and U-net (Convolutional Neural Network).
Tools: Python - PyTorch - Pandas - OpenCV - Google Colab (GPU).
